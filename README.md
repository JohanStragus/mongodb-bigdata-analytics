# 🚀 MongoDB Big Data Analytics
![MongoDB](https://img.shields.io/badge/Built%20with-MongoDB-47A248?style=for-the-badge&logo=mongodb&logoColor=white)

Sistema de análisis de datos en la nube utilizando MongoDB Atlas, enfocado en la monitorización de flotas en tiempo real mediante datos simulados de sensores IoT.

Este proyecto demuestra la implementación de una arquitectura segura (Zero Trust) junto con el uso de Aggregation Pipeline para el procesamiento y análisis de grandes volúmenes de datos.

---

## 🧠 Tecnologías utilizadas

- MongoDB Atlas (Database as a Service)
- MongoDB Compass
- Aggregation Pipeline
- JSON (simulación de datos)
- TLS (comunicación segura)

---

## 🔐 Seguridad implementada

- IP Whitelisting (protección perimetral)
- RBAC (control de acceso por roles)
- Comunicación cifrada mediante TLS
- Modelo de seguridad Zero Trust

---

## 📊 Casos de uso desarrollados

Se han implementado distintos escenarios de análisis sobre los datos:

- Detección de temperaturas críticas en motores  
- Control de la cadena de frío en transporte  
- Auditoría de seguridad en protocolos TLS  
- Análisis de riesgo según velocidad y experiencia del conductor  
- Monitorización de nivel de combustible  
- Identificación de conductores con velocidad media elevada

## 🎯 Objetivo del proyecto

El objetivo principal es demostrar cómo una infraestructura Big Data en la nube puede:

- Procesar grandes volúmenes de datos en tiempo real  
- Detectar riesgos operativos  
- Mejorar la seguridad de los sistemas  
- Facilitar la toma de decisiones basada en datos  

---

## 🚀 Resultados

El sistema permite identificar patrones de riesgo y situaciones críticas en tiempo real, optimizando la eficiencia operativa y mejorando la seguridad en entornos de transporte.
